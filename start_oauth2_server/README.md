OAuth2 Server端配置

    $ keytool -genkeypair -alias jwt -keyalg RSA -dname "CN=jwt, L=Berlin, S=Berlin, C=DE" -keypass mySecretKey -keystore jwt.jks -storepass mySecretKey

将秘钥存储到OAuth2 Server端的src/main/resources/jwt.jks
